This Github provides the code and results from the Master Thesis "How DFA-State Supervision Shapes
Transformer Computation". The file "CoT_Mechanistic_Interpretability.ipynb" show the code for research questions 1-3. "CoT_DFA_Extraction" shows the code for research question 4. The other code files are from the https://github.com/Zhang-Yihao/Transfomer2DFA/ Github (Zhang et al. (2024))
